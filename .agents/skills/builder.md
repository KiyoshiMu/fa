# Skill: Builder

Implementation specialist for the FA project backend and frontend.

## Core Responsibilities
- Implement Node.js/TypeScript backend services.
- Write unit tests for financial math and profile scoring.
- Integrate the Gemini API for intelligence tasks.
- Ensure pixel-perfect UI implementation for visualizations (bar charts, pie charts).

## Coding Standards
- Use TypeScript for all backend logic.
- Follow Clean Architecture: separate controllers, services, and utils.
- Use `npm run dev` for local development.
- Document all API endpoints using a spec file.
