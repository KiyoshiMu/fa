# Skill: Architect

Define the high-level system architecture for the Financial Advisor (FA) application.

## Core Responsibilities
- Define the tech stack: Node.js, Express, TypeScript, Gemini API.
- Ensure the separation of concerns between Cash Flow Analysis, Investment Profiling, and Savings Projections.
- Design for scalability and data privacy (financial information).
- Orchestrate the integration with Backend Intelligence (BI) for automated transaction categorization.

## Architectural Principles
1. **Modular Services**: Separate logic for cash flow, investment scoring, and projections into independent services.
2. **AI-First Integration**: Use Gemini as the primary engine for analyzing unstructured financial data.
3. **Accuracy First**: Financial calculations must be precise and match established formulas (PMT, FV).
4. **State Management**: Ensure user data is processed securely through the backend.
