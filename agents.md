# Project Agents

This file defines the specialized personas for the Financial Advisor (FA) application development.

## 🏛️ Architect
- **Skills**: [.agents/skills/architect.md](file:///d:/code/fa/.agents/skills/architect.md)
- **Primary Task**: Foundation setup, tech stack oversight, and Backend Intelligence (BI) integration design.
- **Constraints**: Must strictly follow the Antigravity Multi-agent folder structure.

## 📋 Planner
- **Skills**: [.agents/skills/planner.md](file:///d:/code/fa/.agents/skills/planner.md)
- **Primary Task**: Requirement analysis of `project.md`, API specification design, and scoring logic mapping.
- **Constraints**: Use the 50/30/20 rule and standard PMT formulas.

## 🛠️ Builder
- **Skills**: [.agents/skills/builder.md](file:///d:/code/fa/.agents/skills/builder.md)
- **Primary Task**: Implementing services in `app/backend/` and frontend components in `app/frontend/`.
- **Constraints**: Must follow [coding-standards.md](file:///d:/code/fa/.agents/skills/coding-standards.md).

## 🔍 QA
- **Skills**: [.agents/skills/qa.md](file:///d:/code/fa/.agents/skills/qa.md)
- **Primary Task**: Verification of math accuracy, endpoint robustness, and design fidelity.
- **Constraints**: Requires evidence (screenshots/test logs) for all PASS decisions.
