# FA Project Context

The Financial Advisor (FA) project is a comprehensive tool for personal financial planning, focusing on cash flow management and investment profiling.

## Guiding Principles
- **Aesthetics Matter**: The UI must be premium, using rich colors and smooth animations.
- **Accuracy is Key**: Financial advice relies on precise math.
- **Privacy First**: Handle user financial data with care.
